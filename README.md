# ✈️ Travel India - Mobile App UI (Figma)

## 📌 Project Overview

Travel India is a modern mobile application UI designed in **Figma** to help users explore popular tourist destinations across India and book travel packages easily. The design focuses on a simple, attractive, and user-friendly interface.

This project was created as a UI/UX design practice to improve skills in mobile app design, layouts, color combinations, and user experience.

---

## 🎯 Features

- 🏖️ Home screen with travel categories
- 📍 Destination selection page
- 🌍 Popular tourist places
- 📝 Booking details form
- 💳 Payment summary screen
- ✅ Booking confirmation page
- 🙏 Thank you screen
